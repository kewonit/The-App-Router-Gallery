# Script Component

Control when third-party scripts load using `next/script`.

- `beforeInteractive` — before page hydration (consent managers, bot detection)
- `afterInteractive` — after the page is interactive (default)
- `lazyOnload` — during browser idle time (chat widgets, analytics)
